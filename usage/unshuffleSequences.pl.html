unshuffleSequences.pl¶

Usage:
shuffleSequences.pl [-p] [-f] < input > output
Options:
-f Format of files (fastq or fasta). [fastq]
-p Output file prefix. [unshuf]
Description:
The script unshuffles Illumina paired-end read files. Reads are assumed to be in order with all pairs intact.
